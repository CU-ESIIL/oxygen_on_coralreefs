# Work Plan

This page tracks the current shape of the oxygen and coral reef project. Use it as a lightweight bridge between meeting notes, code updates, outputs, and the public site.

## Kickoff

- Clarify the main prediction questions around dissolved oxygen, bleaching, reef health, and coral species occurrence.
- Identify core dissolved oxygen datasets and their metadata, temporal coverage, and spatial coverage.
- Record data access decisions in the repository so future collaborators can reproduce the workflow.

## Data Assembly

- Organize raw and derived dissolved oxygen datasets under `data/`.
- Document dataset provenance in the repository's [data README](https://github.com/CU-ESIIL/oxygen_on_coralreefs/blob/main/data/README_data.md).
- Track processing choices in [Data Processing](resources/data_processing.md).
